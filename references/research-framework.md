# 六维信息采集框架 v1.0

> 本文件定义灵感捕手工作流 Phase 1 的标准研究维度。
> 每个维度的调研结果写入对应文件，路径：`{skill-name}/references/research/`

---

## 六个研究维度

| Agent | 维度 | 回答的核心问题 | 输出文件 |
|-------|------|--------------|---------|
| Agent 1 | 著作 | 这个人在系统性思考什么？ | `01-writings.md` |
| Agent 2 | 对话 | 这个人在被追问时怎么反应？ | `02-conversations.md` |
| Agent 3 | 碎片表达 | 这个人下意识怎么说话？ | `03-fragments.md` |
| Agent 4 | 他者视角 | 别人怎么看这个人？ | `04-external.md` |
| Agent 5 | 决策 | 这个人在关键时刻选了什么？ | `05-decisions.md` |
| Agent 6 | 时间线 | 这个人的思想怎么演化的？ | `06-timeline.md` |

## Agent 1：著作维度

**核心问题：这个人系统性思考什么？**

### 搜索方向
- 这个人出版的书籍（书名、核心论点、出版年份）
- 长篇 newsletter / 博客 / 论文
- 反复出现 ≥3 次的核心论点（真信念，不是随口一说）
- 自创术语（这个人自己造的概念）
- 推荐书单（揭示智识谱系）

### 输出格式

```markdown
# 著作维度研究

## 书籍
| 书名 | 年份 | 核心论点 | 代表概念 |
|------|------|---------|---------|
| | | | |

## 反复出现的论点（≥3次出现）
1. **{论点}** — 出现在{来源1}、{来源2}、{来源3}
2. **{论点}** — 出现在{来源1}、{来源2}

## 自创术语
| 术语 | 定义 | 首次出现 |
|------|------|---------|
| | | |

## 推荐书单（揭示智识谱系）
| 书名 | 为什么推荐（体现此人受谁影响） |
|------|--------------------------|
| | |

## 信息可信度评估
- 一手来源：{列出}
- 二手转述：{列出}
- 推测/推断：{列出}
```

## Agent 2：对话维度

**核心问题：此人在被追问时怎么反应？**

### 搜索方向
- 播客访谈、YouTube 长视频、AMA（Ask Me Anything）
- 即兴类比（被追问时的打比方方式）
- 改变立场的瞬间（什么时候承认自己错了）
- 拒绝回答的问题（什么是此人不愿讨论的）

### 输出格式

```markdown
# 对话维度研究

## 即兴类比（此人如何用类比解释复杂问题）
| 类比 | 解释的问题 | 来源 |
|------|-----------|------|
| | | |

## 立场变化记录（此人什么时候改变过观点）
| 事件 | 原立场 | 变化后 | 触发原因 |
|------|--------|--------|---------|
| | | | |

## 拒绝/回避的问题
| 问题 | 此人如何回避 | 来源 |
|------|-----------|------|
| | | |

## 追问反应模式
- 被质疑时：{描述}
- 被追问细节时：{描述}
- 被要求简化时：{描述}

## 信息可信度评估
（同上格式）
```

## Agent 5：决策维度

**核心问题：此人在关键时刻做了什么选择？**

### 搜索方向
- 重大人生/职业决策（离开、加入、放弃、坚守）
- 决策背景和当时的信息环境
- 事后反思（此人事后怎么说自己的决策）
- 言行一致 vs 言行不一案例

|------|------|------|------|---------|---------|
| | | | | | |

|---------|------|

## 并行执行规则

**启动方式：** 同时 spawn 6 个 subagent，每个负责一个维度。

**每个 subagent 的任务 prompt 结构：**
```
你的任务：调研[人物/主题]的{维度名称}。

搜索方向：{从上方对应维度复制}
输出路径：{skill目录}/references/research/0X-{dimension}.md

硬性要求：
- 每条信息标注来源URL和可信度（一手 > 二手 > 推测）
- 区分「此人说的」vs「别人说他的」vs「我推断的」
- 发现矛盾直接记录，不要调和
- 搜索无结果时，写入「信息不足，待补充」

信息源优先级：书籍 > 长对话 > 决策记录 > 社交媒体 > 他人评价 > 二手转述
信息源黑名单：知乎、微信公众号、百度百科（永远不作为来源）
```
