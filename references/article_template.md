# 推广文章模板

> 本模板用于灵感捕手工作流产出配套公众号推广文章。
> 调用 yinyo-writer skill，按其方法论撰写。
> 以下章节为标准结构，不可缺失。

---

## 【封面图片】
「图片」

---

## 【标题】

（由 yinyo-writer 根据具体 skill 内容生成，参考格式：）

---

## 【正文结构】

### 开头：从具体场景切入（不宏大叙事）

（用 yinyo-writer 的开头必杀技之一：叙事启动/荒诞事实/好奇心驱动/感性事件）

### 第一节：这个 skill 是什么

（3-5句话，用「」标点，不用冒号破折号）
（结尾扣回主题）

### 第二节：它解决什么问题

（从具体痛点切入，1-2个具体场景）
（每个痛点配一个"如果你是这种情况"的具体描述）
（结尾升华到方法论）

### 第三节：怎么用

（1-2个具体使用场景，配使用示例）
（操作步骤用「步骤X」格式，不要用序号列表）

### 第四节：为什么这个 skill 值得信任

（从研究过程提炼1-2个独特洞察）
（不要泛泛而谈，要有具体发现）
（呼应开头的事件/场景）

### 结尾：有回响的收尾

（行动指引：附 GitHub 链接）
（结尾句：信念宣言/引用收尾/开放性问题，不写"感谢阅读"）
（固定尾部：点赞/在看/转发三连）

---

## 【标准行动指引格式】

```
GitHub地址（直接复制扔给你的Agent）

{skill-name}：{一句话描述}
https://github.com/xiaoshiyilangzhao1996-droid/{repo-name}
```

---

## 【质检前置检查】（写完文章后自检）

- [ ] 全文无冒号":"（用逗号替代）
- [ ] 全文无破折号"——"
- [ ] 无"说白了/意味着什么/本质上/换句话说"
- [ ] 开头不是宏大叙事
- [ ] 结尾不是"感谢阅读"
- [ ] 有至少一处文化升维
- [ ] 有至少一处第一人称真实经历
- [ ] 行动指引格式符合标准
- [ ] 通过 L4 活人感终审：读起来像一个真实的人在认真聊一件打动自己的事
