# 灵感捕手工作流

> 热点是信号，研究是挖掘，创作是转化。
> 把一个输入，变成一个完整的产品包——skill、日志、文章，三件套。

## 核心理念

一个将热点话题转化为完整技能包的自动化创作流程。

**输入** → **研究 + 发散** → **三件套产出**

不是拿着用户的描述套模板，而是先研究、先发散、先找到真正的 gap，再动手。

## 工作流程

```
第一阶段：输入理解
  接受任意热点素材（链接/地址/事件/描述）
  ↓
第二阶段：研究与发散
  深入研究 → 开拓发散 → 找到创作切入点
  ↓
第三阶段：三件套产出
  skill.skill + 创作日志.md + 推广文章.md
```

## 三件套说明

| 产出物 | 内容 | 用途 |
|--------|------|------|
| `SKILL.md` | 可运行的 Agent Skill | 直接使用 |
| `创作日志.md` | 完整研究+决策记录 | 透明化创作过程 |
| `推广文章.md` | 公众号配套文章 | 吸引用户使用 |

## 安装

```bash
git clone https://github.com/xiaoshiyilangzhao1996-droid/inspiration-hunter.git \

```

## 使用

```
/灵感捕手
```

或直接说：**帮我做个skill / 热点创作 / 从0到1做一个skill**

然后提供你的热点素材（文章链接、GitHub仓库、新闻事件等）
